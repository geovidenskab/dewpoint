# Dugpunktskurve Applikation

En interaktiv webapplikation til at visualisere og beregne relativ luftfugtighed baseret på temperatur og dugpunkt.

## Funktioner

- **Interaktiv graf:** Viser mætningskurven for vanddamp i luft
- **Input felter:** Indtast temperatur og dugpunkt
- **Visuelle linjer:** Røde og grønne linjer der viser aflæsninger
- **Pædagogisk animation:** Trin-for-trin forklaring af beregningen
- **KaTeX formler:** Pæne matematikformler med rigtige brøker
- **Dansk formatering:** Komma som decimal-separator

## Tekniske detaljer

- **React.js:** Frontend framework
- **Plotly.js:** Interaktive grafer
- **KaTeX:** Matematikformler
- **Tailwind CSS:** Styling
- **Magnus formel:** Fysisk korrekt beregning af mætningsdampstryk

## Installation og kørsel

```bash
npm install
npm start
```

## Bygning til produktion

```bash
npm run build
```
